What is NieR?

Nier is an innovative Japanese role-playing game (JRPG) franchise blending fantasy and sci-fi elements created by game designer Yoko Taro. The first NieR game was released in 2010. Since then, two more entries and a remaster have been created, amassing a community of dedicated fans.
